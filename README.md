# Logística 2V / V2

📌 Sobre o Projeto

O Logística 2V / V2 é um sistema web voltado para logística reversa, atendendo às necessidades de substituições e devoluções de pequenos comerciantes. Muitos desses comerciantes utilizam marketplaces ou redes sociais para vendas e não possuem uma estrutura formal para gerenciar esses processos.

Nosso objetivo é garantir simplicidade de uso para consumidores e lojistas, permitindo o gerenciamento eficiente de solicitações e feedbacks. Com isso, buscamos aumentar a precisão das informações e auxiliar na tomada de decisões corretivas baseadas nos dados gerados pelo sistema.

**Backend**

**Banco de Dados**

**Intermediador entre Frontend e Backend**

* Node.js versão 22.14.0 (LTS)

📦 Dependências do Spring Boot

* Spring Web: Para construção de APIs RESTful.
* Spring Data JPA: Gerenciamento de dados com ORM.
* Lombok: Redução de código boilerplate.
* Swagger 2.0.3: Documentação da API.
* H2 Database: Banco em memória para testes.
* JWT Api / Impl / Jackson (v0.11.5): Autenticação e manipulação de JSON.
